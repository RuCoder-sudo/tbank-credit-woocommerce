## Профессиональный платёжный шлюз WooCommerce для рассрочки и кредита через Т-Рассрочка от Т-Банка

**Т-Банк Рассрочка и Кредит для WooCommerce** — это полноценный платёжный шлюз, который добавляет на ваш сайт возможность оплаты товаров в **рассрочку** и **кредит** через сервис **Т-Рассрочка** (Т-Банк). Реальная интеграция через REST API, кнопки на всех страницах, AJAX-калькулятор, email-уведомления, промо-коды, период охлаждения и тестовый режим.

---

### 🚀 Основные возможности

#### 💰 Приём платежей
- **Рассрочка** — оплата частями без переплаты.
- **Кредит** — кредитование через Т-Банк.
- Полная интеграция с REST API Т-Рассрочка.

#### 🧮 AJAX-калькулятор
- Интерактивный расчёт ежемесячного платежа.
- Отображение на странице товара, корзине и оформлении заказа.
- Адаптивность и кастомизация стилей.

#### 🔘 Кнопки на всех страницах
- Кнопка «Купить в рассрочку» на странице товара.
- Кнопки в каталоге, корзине, виджетах.
- Возможность изменить текст и дизайн кнопки.

#### 📧 Email-уведомления
- Автоматические письма клиенту о статусе заказа (рассрочка/кредит).
- Настраиваемые шаблоны.

#### 🎟️ Промо-коды и период охлаждения
- Встроенная поддержка промо-кодов на рассрочку.
- Настройка периода охлаждения — минимальное время между повторными заявками.

#### 🧪 Тестовый режим
- Полная эмуляция платежей для отладки.
- Возможность включить логирование всех запросов к API.

#### 📊 Административная панель
- Просмотр всех транзакций рассрочки/кредита.
- Фильтрация по дате, сумме, статусу.
- Возможность вручную изменить статус платежа.

### 📥 Установка

1. Скачайте архив плагина или клонируйте репозиторий:
   ```bash
   git clone https://github.com/RuCoder-sudo/tbank-credit-woocommerce.git
   ```
2. Загрузите папку `tbank-credit-woocommerce` в `/wp-content/plugins/`.
3. Активируйте плагин через меню **Плагины** в WordPress.
4. Перейдите в **WooCommerce → Настройки → Платежи**.
5. Активируйте метод «**Т-Банк Рассрочка и Кредит**».
6. Введите **API-ключи** (получаются в личном кабинете Т-Банка → Т-Рассрочка).
7. Настройте кнопки, калькулятор, промо-коды и период охлаждения.

> 💡 **Совет:** после активации включите тестовый режим и проверьте работу калькулятора и оформления заказа.

### ❓ Часто задаваемые вопросы

**Где взять API-ключи Т-Банка?**  
Заключите договор с Т-Банком на подключение сервиса **Т-Рассрочка**. После этого в личном кабинете партнёра (раздел «API») вы получите `client_id` и `api_key`.

**Нужен ли отдельный договор для рассрочки и кредита?**  
Обычно достаточно одного договора на сервис «Т-Рассрочка», который включает оба продукта.

**Как настроить промо-коды?**  
В настройках плагина есть раздел «Промо-коды». Вы можете создать неограниченное количество кодов с разными типами скидок (процент, фиксированная сумма, бесплатная доставка).

**Что такое период охлаждения?**  
Это временной интервал, который должен пройти после отказа или завершения предыдущей заявки на рассрочку/кредит, прежде чем пользователь сможет подать новую заявку. По умолчанию 24 часа.

**Как изменить внешний вид кнопки или калькулятора?**  
Плагин автоматически наследует стили вашей темы. Для более тонкой настройки используйте фильтры и хуки (документация в файле `docs/customization.md`).

**Работает ли с мультивалютными плагинами?**  
Да, плагин корректно обрабатывает валюту, установленную в WooCommerce (рубли, доллары, евро – конвертация в рубли по курсу Т-Банка).
