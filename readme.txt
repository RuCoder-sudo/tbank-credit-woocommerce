Платёжный шлюз WooCommerce для оформления покупок в рассрочку или кредит через Т-Рассрочка от Т-Банка.

== Description ==

Плагин добавляет платёжный шлюз в WooCommerce, позволяя покупателям оформлять покупки в рассрочку или кредит через систему «Т-Рассрочка».

**Возможности:**

* Платёжный метод в списке WooCommerce с полными настройками
* Поддержка тестового (sandbox) и рабочего (production) режима
* Информер на странице товара с расчётом ежемесячного платежа
* Кнопки «Купить в кредит» / «Купить в рассрочку» в каталоге
* Поддержка промо-кодов (для рассрочки и различных условий кредита)
* Автоматическое обновление статуса заказа через Webhook
* Полная инструкция по подключению прямо в настройках плагина
* Скрытие метода оплаты при сумме ниже 3 000 ₽
* Проверка страны покупателя (только Россия)
* Логирование для отладки

**Как подключить:**

1. Зарегистрируйтесь как партнёр на сайте Т-Банка
2. Получите Shop ID и Showcase ID от менеджера банка
3. Введите данные в настройках плагина
4. Включите тестовый режим, проверьте работу
5. Переключитесь в рабочий режим

== Installation ==

1. Загрузите папку `tbank-credit-woocommerce` в `/wp-content/plugins/`
2. Активируйте плагин в меню «Плагины»
3. Перейдите в WooCommerce → Настройки → Платежи → Т-Банк Рассрочка и Кредит
4. Введите ваши Shop ID и Showcase ID
5. Настройте параметры под ваши нужды
6. Включите плагин и протестируйте

== Frequently Asked Questions ==

= Где взять Shop ID и Showcase ID? =

Эти данные предоставляет менеджер Т-Банка после регистрации вашего магазина в системе кредитования.

= Какая минимальная сумма для оформления кредита? =

3 000 рублей. При меньшей сумме метод оплаты автоматически скрывается.

= Как настроить уведомления (webhook)? =

Войдите в Личный кабинет Т-Банка → «Кредитование в магазинах» → выберите магазин → «Редактировать» → «Уведомления» → укажите URL вашего сайта (например https://mysite.com) и поставьте галочку.

= Как работает тестовый режим? =

В тестовом режиме запросы отправляются с параметром demo=true. Настоящие платежи не проводятся.
